## ✨ Features
- 🔐 **Complete Authentication** - Register/Login with OTP
- 🔍 **AI-Powered Search** - Voice search with smart suggestions
- 📱 **8 Service Categories** - Transport, Tech, Emergency, etc.
- 🚨 **Emergency Features** - Panic button with GPS
- 📲 **PWA Installation** - Add to home screen
- 🌐 **Offline Support** - Works without internet
- 📱 **Mobile Optimized** - Responsive design

## 🔧 Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js/Express API
- **Database**: PostgreSQL with Prisma
- **PWA**: Service Workers, Web App Manifest
- **Authentication**: JWT with OTP verification
- **Voice**: Web Speech API
- **Maps**: Geolocation API

## 🚀 Deployment
- **Live Backend**: https://zippup-backend-v3.onrender.com
- **Frontend**: Deployed on Netlify/Vercel
- **PWA**: Installable on mobile devices
